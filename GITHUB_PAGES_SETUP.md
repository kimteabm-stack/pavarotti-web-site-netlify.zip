# 학교에서 발표 웹페이지 여는 방법

이 저장소에는 발표용 웹페이지 `index.html`이 들어 있습니다.

## GitHub Pages 켜기

1. GitHub에서 이 저장소로 들어갑니다.
2. `Settings`를 누릅니다.
3. 왼쪽 메뉴에서 `Pages`를 누릅니다.
4. `Build and deployment`에서 `Deploy from a branch`를 선택합니다.
5. Branch는 `main`, 폴더는 `/ (root)`로 선택합니다.
6. `Save`를 누릅니다.

몇 분 뒤 보통 아래 주소에서 열립니다.

https://kimteabm-stack.github.io/pavarotti-web-site-netlify.zip/

## 학교에서 열기 전 확인

이 저장소가 private이면 학교 컴퓨터에서 안 열릴 수 있습니다. 그 경우 저장소를 public으로 바꾸거나, GitHub Pages 주소가 외부에서 열리는지 미리 확인하세요.

발표 화면에서는 오른쪽 화살표, 아래 화살표, 스페이스바로 다음 슬라이드로 넘길 수 있습니다. `전체` 버튼을 누르면 전체 화면으로 볼 수 있습니다.
